The Fairview Web Team is a student-run organization that manages and maintains the Fairview High School website.

We meet every Friday after school at 3:30pm. Anyone, regardless of programming experience, is welcome to join.

The Web Team hosts a few events every year, including the Computer Science Education Week.

This website is still being updated with content. Please bear with us!


Technology

The Fairview website is mainly written using Ruby on Rails.
